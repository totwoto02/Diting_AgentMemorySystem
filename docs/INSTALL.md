# MFS 一键安装指南

## 快速开始

### 1. 安装

```bash
pip install mfs-memory
```

### 2. 验证安装

```bash
mfs-check-install
```

### 3. 使用

安装完成后，在 OpenClaw 中自动识别以下工具：

- **mfs_read**: 读取记忆（自动还原切片）
- **mfs_write**: 写入记忆（长文本自动切片）
- **mfs_search**: 搜索记忆（知识图谱扩展）

---

## 自动功能

### 自动切片

写入超过 2000 字的长文本时，自动切分为 500-2000 字的切片：

```
用户写入 5000 字文档
    ↓
自动切分为 3-5 个切片
    ↓
存储到 MFT（带 lcn_pointers 指针）
```

### 自动还原

读取有切片的记忆时，自动拼装还原为完整文本：

```
用户读取记忆
    ↓
根据 lcn_pointers 捞取所有切片
    ↓
自动拼装为完整原文
    ↓
返回给用户
```

### 自动构建知识图谱

写入记忆时自动提取关键词并构建关联：

```
用户写入："用户朋友喜欢游戏，特别是游戏角色"
    ↓
自动提取关键词：["用户朋友", "游戏", "游戏角色"]
    ↓
构建关联：用户朋友→游戏，游戏→游戏角色
    ↓
搜索"用户朋友"时提示："可能也关心：游戏、游戏角色"
```

## 故障排除

### 问题：mfs-check-install 命令不存在

**解决**：
```bash
# 重新安装
pip uninstall mfs-memory
pip install mfs-memory
```

### 问题：MCP 未注册到 OpenClaw

**解决**：
1. 检查 OpenClaw 配置文件是否存在
2. 手动添加 MFS MCP Server 配置
3. 重启 OpenClaw Gateway

### 问题：Python 版本过低

**要求**：Python 3.11+

**解决**：
```bash
# 检查版本
python3 --version

# 升级 Python（根据系统选择）
```
